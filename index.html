About:

Memory, also known as Match Match, is a counter card game where the object is to find pairs.

To Play:

1. Select two cards to try to match the pictures.

2. If you match the pictures you can go again.

3. If they don't match, computer will turn them back.

4. The player that finds all pairs faster than anybody else wins.

5. Have Fun!.
